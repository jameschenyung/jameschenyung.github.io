Creative 5 Software Engineer Internship



During my first internship at Creative 5, my task was to create a live simulation of the electrical output of solar panels given a specific location to test our product since we did not have solar panels in the office. I was able to query the solar irradiance of a location given its coordinates and convert those numbers into a DC Power supply unit which outputed specific values of Voltage and Ampere into our company product (monitoring device). My simulation was run on an SMG (Smart Machine Gateway) which contains a microprocessor provided by Raspberry Pi's compute module. Furthermore, when the Senior Software Engineers were done writing their code to monitor and interact with the whole system, I created a program to flash multiple microprocessors simultaneously for mass production so that their code would be flashed into multiple SMG which allows the code to interact with hardware. During our mass production, we had a factory produce a bunch of IO boards, thus, I had created an automated tester that ensures the quality of the IO board (ports are working, wifi signals, etc.).

Skills used:

  • Python, HTML, Javascript, CSS
  • git and terminal commands

Skills learned:

  • Forced to learn how to use VIM editing tool (IDE) so I could write code directly into the microprocessor.
  • How to communicate with a DC Power Supply Unit with reading and writing using Python Serial.
  • Learned how to make API calls.
